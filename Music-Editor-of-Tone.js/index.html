插件的注意事項

  1. filter只能加在每個區塊的開始
  2. filter的影響範圍限定在自身所在的區塊
  3. plugin lag會使"同區塊"後面的base插件撥放時間後延
  4. block lag只能加在每個區塊的結尾
  5. block lag會使"後面區塊"的base插件撥放時間後延

base中的名詞解釋

Envelope
定義音量如何隨時間改變

  • Attack :
    音量從0到最大值花的時間
  • Decay :
    音量到最大值後，下降至Sustain所花的時間
  • Sustain :
    第二音量的大小
  • Release :
    從第二音量降至0所花的時間
Oscillator

定義音頻的組成，會影響音色
Musical Notation

就只是樂譜


  • BPM :
    一分鐘有幾拍

  • Length Per Duration :
    音長的單位

  • Duration :
    一個音的長度，本程式以Length Per Duration為單位

檢討與未來

目前缺少依照音階調整分貝的功能

這會導致低音與高音的音量差距太大

雖然可以用不同插件區隔出不同的音階

但終究不是很方便

還有聲音聽起來都蠻僵硬的

未來可能會選擇增加預設音色

像是原本就配好的鋼琴、鼓聲等等

操作介面也要改成節點式
