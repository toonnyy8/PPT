宣告

<Declare> -> id

<Declare> -> id , <Declare>

<Declare> -> <Assign>

賦值

<Assign> -> id = <Assign>

<Assign> -> id = <Func>

目前完成的語法-中級篇

  • While Loop
  • If else if else

While

<While> -> while ( <Exp> ) { }

<While> -> while ( <Exp> ) { <Sections> }

If else if else

<If> -> if ( <Exp> ) { }

<If> -> if ( <Exp> ) { } else { }

<If> -> if ( <Exp> ) { } else <If>

<If> -> if ( <Exp> ) { } else { <Sections> }

If else if else

<If> -> if ( <Exp> ) { <Sections> }

<If> -> if ( <Exp> ) { <Sections> } else { }

<If>
-> if ( <Exp> ) { <Sections> } else <If>

<If>
-> if ( <Exp> ) { <Sections> } else { <Sections> }

目前完成的語法-中級上去一點點篇

  • 不能取得外部變數的 Function

Function

<Func> -> id ( )=> { }

<Func> -> id ( )=> { <Sections> }

<Func> -> id ( <Params> )=> { }

<Func> -> id ( <Params> )=> { <Sections> }
