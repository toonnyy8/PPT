關於AI玩格鬥遊戲這檔事

為什麼專題會製作讓深度學習AI玩格鬥遊戲?

一般格鬥遊戲上，常有入門門檻過高與玩家間常出現嚴重的技術斷層等等問題，新手玩家連初級AI都打不贏，而老玩家面對高階AI卻依舊取得壓倒性勝利，這樣巨大的斷層會帶給格鬥遊戲的初學者巨大的挫敗感。

因此，本專題藉由強化學習方法訓練出能用於格鬥遊戲的AI，並利用調整獎勵來製作出不同風格、強度的模型，使得各種玩家都能充分享受格鬥遊戲帶來的樂趣。

插件的注意事項

  1. filter只能加在每個區塊的開始
  2. filter的影響範圍限定在自身所在的區塊
  3. plugin lag會使"同區塊"後面的base插件撥放時間後延
  4. block lag只能加在每個區塊的結尾
  5. block lag會使"後面區塊"的base插件撥放時間後延

base中的名詞解釋

Envelope
定義音量如何隨時間改變

  • Attack :
    音量從0到最大值花的時間
  • Decay :
    音量到最大值後，下降至Sustain所花的時間
  • Sustain :
    第二音量的大小
  • Release :
    從第二音量降至0所花的時間
Oscillator

定義音頻的組成，會影響音色
Musical Notation

就只是樂譜


  • BPM :
    一分鐘有幾拍

  • Length Per Duration :
    音長的單位

  • Duration :
    一個音的長度，本程式以Length Per Duration為單位
